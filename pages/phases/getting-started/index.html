Core SMEQA Roles

HR Specialist

HR Specialists will ____. You should have at least two HR specialists assigned to support the action, though one can be a backup in case of unexpected outage by the primary specialist.

Project Manager

The project manager can be an HR specialist, but does not need to be. This role does tasks like scheduling trainings, keeping track of SMEs and hiring managers, creating template documents, and facilitaing interview scheduling. Project managers should be:

  • Highly organized.
  • Able to manage meeting/collaboration technology
